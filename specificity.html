Specificity Calculation

Specificity is calculated as follows:

  • Element selector (p): Specificity = 0,0,0,1
  • Class selector (.info): Specificity = 0,0,1,0
  • ID selector (#warning): Specificity = 0,1,0,0
  • Combined (#warning.info): Specificity = 0,1,1,0

Higher specificity values override lower ones. When specificity is equal, the last rule in the cascade wins.
